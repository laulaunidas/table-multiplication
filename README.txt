Table des Multiplications

Construire une table des multiplications en JavaScript puis l'afficher en HTML.

Détails
 l'utilisateur saisi la taille de la table des multiplications (exemple : si on saisit 10 il faut faire une table de 1 à 10).
Pour l'affichage, lorsque le nombre est multiplié par lui-même (1x1, 2x2, 3x3, etc.), la cellule du tableau HTML s'afficher d'une autre couleur que les autres cellules. La solution doit être en CSS.
